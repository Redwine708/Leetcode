# 题目大意
```
给你一个链表的头节点 head 和一个特定值 x ，请你对链表进行分隔，使得所有 小于 x 的节点都出现在 大于或等于 x 的节点之前。
你应当 保留 两个分区中每个节点的初始相对位置。
示例 1：
输入：head = [1,4,3,2,5,2], x = 3
输出：[1,2,2,4,3,5]
示例 2：
输入：head = [2,1], x = 2
输出：[1,2] 
提示：
链表中节点的数目在范围 [0, 200] 内
-100 <= Node.val <= 100
-200 <= x <= 200
Link: https://leetcode-cn.com/problems/partition-list/
```
# 解题思路
```
设置两个链表，一个链表(smallList)存储所有小于k的值，另一个链表(largeList)存储大于k的值
然后合并两个链表 smallList -> next = largeListHead->next

    注意特别判断的情况
要将largeList处理结束的下一个指针设置为NULL，否则就会死循环
* 因为我们是没有额外空间开销的，所以如果不特别处理就会出现环链表
``` 
# 复杂度分析
```
时间复杂度 ： T（n） 只用遍历一次链表就可以
空间复杂度 ： O (1)  只使用了两个虚拟头结点

```
